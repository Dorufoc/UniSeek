# 用户协议与隐私政策功能实施计划

## 任务概述
在 Vue 登录页的登录按钮上方增加用户协议和隐私政策的勾选项，勾选后才能登录。用户协议和隐私政策的文本标签能够点击跳转到独立页面，并根据项目整体设计风格完成这两个页面的设计和开发。

## 当前状态分析

### 项目技术栈
- **框架**: Vue 3 + TypeScript + Vite
- **UI 组件库**: Element Plus
- **路由**: Vue Router 4
- **状态管理**: Pinia

### 登录页现状
- **文件位置**: `uniseek_vue/src/pages/Login.vue`
- **现有结构**:
  - 登录/注册选项卡切换
  - 手机号、密码输入框
  - 登录按钮
  - 第三方登录入口（预留）
- **已有样式**: CSS 中已定义 `.form-options` 和 `.agree-label` 类，但模板中未使用

### 设计风格特征
- **主色调**: `#1762FB`（蓝色）
- **背景渐变**: `linear-gradient(135deg, #e8f4fd 0%, #d6ebfa 50%, #cce5ff 100%)`
- **卡片设计**: 圆角 12px，阴影 `0 1px 4px rgba(0,0,0,0.06)`
- **页面背景**: `#f5f7fa`
- **字体**: 标题 22px，正文 14-15px
- **间距**: 组件间距 16-24px

## 实施步骤

### 步骤 1: 修改登录页 Login.vue
**文件**: `uniseek_vue/src/pages/Login.vue`

**修改内容**:
1. 在 `<script setup>` 中添加协议勾选状态
   ```typescript
   // 用户协议和隐私政策勾选状态
   const agreed = ref(false)
   ```

2. 修改登录按钮的禁用条件
   ```typescript
   const canLogin = computed(() => {
     return isPhoneValid.value && isPasswordValid.value && agreed.value
   })
   ```

3. 在登录按钮上方添加勾选框 UI
   - 位置：在 `<!-- 登录按钮 -->` 之前
   - 包含勾选框 + "我已阅读并同意" 文本 + 用户协议链接 + "和" + 隐私政策链接
   - 使用 Element Plus 的 `el-checkbox` 组件
   - 链接使用 `<router-link>` 跳转到对应页面

4. 添加相关样式
   - 使用已定义的 `.agree-label` 样式
   - 链接颜色使用主色调 `#1762FB`

### 步骤 2: 创建用户协议页面
**文件**: `uniseek_vue/src/pages/UserAgreement.vue`

**页面设计**:
- **布局**: 与 AccountSecurity.vue 保持一致的卡片式布局

  - 副标题：请仔细阅读以下条款

**内容大纲**:

4. 服务内容与限制

8. 争议解决

**样式设计**:
- 页面背景：`#f5f7fa`
- 内容卡片：白色背景，圆角 12px，阴影
- 标题：22px，加粗

### 步骤 4: 配置路由
**文件**: `uniseek_vue/src/router/index.ts`

**添加路由**:
```typescript
{
  path: '/user-agreement',
  name: 'UserAgreement',
  component: () => import('@/pages/UserAgreement.vue'),
  meta: { title: '用户协议 - UniSeek' }
},
{
  path: '/privacy-policy',
  name: 'PrivacyPolicy',
  component: () => import('@/pages/PrivacyPolicy.vue'),
  meta: { title: '隐私政策 - UniSeek' }

```

**路由位置**: 与 `/login` 路由同级，不需要登录认证
