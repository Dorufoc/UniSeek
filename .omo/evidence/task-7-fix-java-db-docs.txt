### 最终 selectEnterpriseTasks 查询（lines 167-180）：
```sql
SELECT
    <selectTaskFields/>,
    COALESCE(apps.c, 0) AS application_count
FROM task
LEFT JOIN enterprise ON task.enterprise_id = enterprise.id
LEFT JOIN category ON task.category_id = category.id
LEFT JOIN region ON task.region_id = region.id
LEFT JOIN (
    SELECT task_id, COUNT(*) AS c
    FROM task_application
    GROUP BY task_id
) AS apps ON task.id = apps.task_id
WHERE task.enterprise_id = #{enterpriseId}
ORDER BY task.create_time DESC
```
